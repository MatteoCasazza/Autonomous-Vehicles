# Autonomous Vehicles Projects

This repository presents hands-on implementations of core algorithms used in autonomous driving systems, developed through a series of assignments and projects. For more information about all the work, take a look to the report [Autonomous_Vehicles_report.pdf](Autonomous_Vehicles_report.pdf) 

## 📌 Overview
The work covers the main components of autonomous navigation:
- Perception and sensor fusion  
- Feedback control and trajectory tracking  
- Motion planning (grid-based and sampling-based)  
- State machines and system modeling  
- SLAM-based navigation in realistic environments  

## 📂 Repository Structure

- [Assignment 1 – Trajectory Reconstruction & Mapping](./Assignment_1)  
  Reconstruction of robot trajectory from ROS2 bag data and environment mapping using LiDAR.

- [Assignment 2 – Waypoint Navigation & Control](./Assignment_2)  
  Implementation and comparison of multiple feedback controllers for waypoint tracking.

- [Assignment 3 – Perception-Based Target Reaching](./Assignment_3)  
  Sensor fusion (camera + LiDAR) for target detection and autonomous navigation.

- [Assignment 4 – Grid-Based Path Planning](./Assignment_4)  
  Implementation of Dijkstra and A* algorithms with different heuristics.

- [Assignment 5 – Finite State Machine for Parking System](./Assignment_5)  
  Design and simulation of a vehicle–gate interaction system with physical constraints.

- [Project 1 – Sampling-Based Motion Planning (RRT)](./Project_1)  
  Implementation and comparison of RRT, RRT-Connect, and RRT* algorithms.

- [Project 2 – SLAM-Based Autonomous Navigation](./Project_2)  
  Integration of SLAM, path planning, and control in a realistic indoor environment.
